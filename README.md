# Recipe CRUD API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Installation

1. Clone repository
2. Run npm install
3. Create .env file
4. Run npm run dev

## Environment Variables

PORT= 5002
MONGO_URL = mongodb://127.0.0.1:27017/recipeApp

## API Endpoints

POST /api/recipeapp
GET /api/recipeapp
GET /api/recipeapp/:id
PUT /api/recipeapp/:id
DELETE /api/recipeapp/:id

## Deployment

Deployed on Render
