# Recipe CRUD API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Installation

1. Clone repository
2. Run npm install
3. Create .env file
4. Run npm run dev

## Environment Variables

PORT=5002
MONGO_URI = mongodb://127.0.0.1:27017/recipeApp

## API Endpoints

POST /api/recipes
GET /api/recipes
GET /api/recipes/:id
PUT /api/recipes/:id
DELETE /api/recipes/:id

## Deployment

Deployed on Render
