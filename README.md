# Recipe CRUD API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Installation

1. Clone repository
2. Run npm install
3. Create .env file
4. Run npm run dev

## Environment Variables

PORT=5002
MONGO_URI=your_mongodb_connection_string

## API Endpoints

POST /api/recipeapp
GET /api/recipeapp
GET /api/recipeapp/:id
PUT /api/recipeapp/:id
DELETE /api/recipeapp/:id

## Deployment
